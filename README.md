# Core Inventory Management System

A robust and modern Inventory Management System (IMS) built with Next.js, React, and Prisma.

## Features
- Real-time stock tracking and allocation.
- Comprehensive warehouse management.
- Move history and activity logging.
- Advanced search and filtering.
- Responsive dashboard with data visualization.

## Tech Stack
- **Frontend**: Next.js, Tailwind CSS, Lucide Icons.
- **Backend**: Prisma (with MongoDB), Next.js API Routes.
- **State Management**: TanStack Query (React Query) and Zustand.
- **Validation**: Zod.

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up your environment variables in `.env`.
4. Run the development server: `npm run dev`.
5. Open [http://localhost:3000](http://localhost:3000) in your browser.
